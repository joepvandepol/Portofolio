Hobby project

I have always been fascinated by blockchain. That's why I'm building my own blockchain in Node.js. The blockchain works by connecting sockets(P2P). After a socket it connected it will sync up to the latest block and you're ready to go.

This project isn't finished yet and the project is a private repo on Github. If you'd like to see the code send me an e-mail and I will invite you to the repo.

  • 01. API
  • 02. Features
  • 03. Development

The following API calls can be made:

  1. Retrieving wallet/balance(explorer)
  2. Requesting a new public and private key
  3. Create transaction
  4. Retrieve all mined blocks
  5. Mine blocks but this wil be costly because if there are new blocks the blocks will be broadcasted to all the sockets that are connected

The Node.js blockchain has all the default features of a blockchain. The blockchain mining systeem is a hybrid between POS and POW. Every wallet holder can buy up to 3 tickets. With these tickets the wallet holder can approve or decline a mined block. The blockchain aims to sell about 50.000 tickets. The price of a ticket is calculated by the amount of tickets that are bought.
After the miner mines a block there will be 3 randomly chosen tickets to approve or decline the newly mined block. There must be at least 2/3 approval.


I'm currently working on the CLI for generating/importing wallets, sending transaction and checking balance. All features are currently working through Postman. With the CLI version you can easier manage your wallet.
